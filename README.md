## Prerequisites
Before running the application, ensure you have the following installed:

 * Java Development Kit (JDK) 21 or later
 * Maven 3.6 or later
 * JavaFX SDK 21
 * An IDE (IntelliJ IDEA or VS Code recommended)

## Technology Stack

## Backend:

 * Spring Boot 3.x
 * Spring Data JPA
 * H2 Database
 * RESTful APIs


## Frontend:

 * JavaFX 21
 * Scene Builder
 * CSS for styling
